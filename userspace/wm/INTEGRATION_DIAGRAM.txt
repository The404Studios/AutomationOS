Integration Points Summary
===========================

✅ Window Lifecycle:
   • Creation: WM → Compositor (IPC_CREATE_WINDOW)
   • Mapping: WM → Compositor (IPC_MAP_WINDOW)
   • Focus: WM → Compositor (IPC_FOCUS_WINDOW)
   • Destruction: WM → Compositor (IPC_DESTROY_WINDOW)

✅ Geometry Management:
   • Move: compositor_sync_geometry()
   • Resize: compositor_sync_geometry()
   • Maximize: compositor_sync_geometry() + state update
   • Tiling: batch geometry updates

✅ Visual Rendering:
   • Decorations: compositor_render_decorations()
   • Z-order: wm_get_window_z_order()
   • Damage: wm_notify_geometry_change()

✅ Input Handling:
   • Mouse: Compositor → WM (IPC_MOUSE_EVENT)
   • Keyboard: Compositor → WM (IPC_KEYBOARD_EVENT)
   • Shortcuts: WM processes, updates windows
   • Forwarding: WM → App (for focused window)

✅ State Synchronization:
   • Full sync: Every 60 frames
   • Incremental: On every change
   • Recovery: Automatic on reconnect
