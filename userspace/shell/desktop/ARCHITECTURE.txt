================================================================================
                            THEME SYSTEM
================================================================================

theme_t
├── Colors
│   ├── Primary (accent)
│   ├── Backgrounds (primary, secondary, tertiary)
│   ├── Text (primary, secondary, tertiary)
│   └── UI Elements (panel, dock, window, separator)
│
├── Effects
│   ├── Blur radius
│   ├── Shadow opacity
│   └── Corner radius
│
└── Typography
    ├── Font family
    └── Font sizes (small, body, heading)

Light Theme          Dark Theme           Auto Theme
├── White BG         ├── Dark BG          ├── 6am-6pm: Light
├── Black Text       ├── White Text       └── 6pm-6am: Dark
└── Blue Accent      └── Blue Accent

================================================================================
                            IPC INTEGRATION
================================================================================

Applications ──────IPC Messages────> Desktop Shell
                                     │
                                     ├─ MSG_NOTIFY → notification_center
                                     ├─ MSG_DOCK_ADD → dock
                                     ├─ MSG_PROGRESS → dock (progress bar)
                                     ├─ MSG_WINDOW_CREATE → window manager
                                     └─ MSG_TRAY_ADD → panel (system tray)

================================================================================
                            RENDER ORDER (Z-Index)
================================================================================

Layer 0: Desktop (wallpaper + icons)
Layer 1: Windows (application windows)
Layer 2: Dock (always above windows)
Layer 3: Panel (always on top)
Layer 4: Overview (fullscreen overlay)
Layer 5: Quick Settings (slide-in panel)
Layer 6: System Menu (dropdown menu)
Layer 7: Notification Center (slide-in panel)
Layer 8: Notification Toasts (temporary popups)
